# Technical Documentation & Consulting

## Resumen

Repositorio que demuestra **capacidad de documentación y consultoría técnica**.  
Incluye plantillas, procedimientos y guías para proyectos IT y de ciberseguridad.

---

## Contenido destacado

- Documentos de análisis de sistemas y seguridad  
- Procedimientos operativos estándar  
- Plantillas de informes para dirección y clientes  
- Buenas prácticas en comunicación técnica

---

## Lo que muestra a los reclutadores

- Habilidad para comunicar soluciones complejas  
- Experiencia en documentación profesional y consultoría  
- Capacidad de estructurar información para diferentes niveles de stakeholders
