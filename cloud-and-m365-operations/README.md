# Cloud & Microsoft 365 Operations

## Resumen

Repositorio orientado a la **gestión de Microsoft 365 y entornos cloud**, incluyendo licenciamiento, soporte y seguridad.

---

## Contenido destacado

- Procedimientos de gestión de licencias y cuentas  
- Buenas prácticas en O365 y cloud  
- Seguridad y cumplimiento en entornos corporativos  
- Automatización básica de tareas y reporting

---

## Lo que muestra a los reclutadores

- Dominio de herramientas Microsoft 365 y cloud  
- Capacidad de soporte operativo y mejora continua  
- Aplicación de estándares de seguridad y compliance


## Flujo de Gestión O365

```text
Usuarios --> Portal O365 --> Licencias & Roles --> Aplicaciones (Teams, Exchange, SharePoint)
                                   │
                                   ▼
                               Logs & Auditoría
                                   │
                                   ▼
                            Seguridad & Compliance
```
